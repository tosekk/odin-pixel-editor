
# Odin + Raylib Pixel Editor

A simple pixel editor built using [Odin programming language](https://odin-lang.org/) and [raylib](https://www.raylib.com/).

# Features

- [x] Pen
- [x] Eraser
- [x] Fill bucket
- [x] Adobe Photoshop-like color switching
- [ ] Color selection
- [ ] Custom canvas sizes
- [x] Zoom in/out
- [ ] Export image
- [ ] Undo/redo
