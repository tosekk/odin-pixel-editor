
# Odin + Raylib Pixel Editor

A simple pixel editor built using [Odin programming language](https://odin-lang.org/) and [raylib](https://www.raylib.com/).

# Features

- Pen
- Eraser
- Fill bucket
- Adobe Photoshop-like color switching
- Color selection
- Custom canvas sizes
- Zoom in/out
- Export image
- Undo/redo
